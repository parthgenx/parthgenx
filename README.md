## Featured Projects

<details>
<summary><b>&nbsp;⬡&nbsp; IntelliDoc — AI-Powered Document Intelligence Platform</b></summary>
<br/>

> An enterprise-grade document intelligence platform enabling conversational AI interactions with PDF content — built from first principles with a production-ready RAG architecture.

<div align="center">

| Attribute | Details |
|:---|:---|
| **Stack** | React.js · FastAPI · Anthropic Claude API · Pinecone · Supabase |
| **AI Core** | RAG pipeline with 768-dimensional vector embeddings · Top-3 semantic retrieval |
| **Security** | Supabase Auth · Row Level Security (RLS) · Complete tenant data isolation |
| **Performance** | Sub-second query responses via Pinecone ANN vector search |
| **Impact** | Conversational interface enabling natural language queries over private documents |
| **Repository** | [![GitHub](https://img.shields.io/badge/View%20Repo-171515?style=flat-square&logo=github&logoColor=white)](https://github.com/parthgenx/Intellidoc) |

</div>

IntelliDoc is a full-stack web application where users upload PDF documents and interact with their content through a conversational AI interface. The core engine is a custom-built RAG pipeline: documents are chunked, converted into 768-dimensional vector embeddings via the Anthropic Claude API, and stored in Pinecone. At query time, the system retrieves the top 3 semantically relevant sections and feeds them as grounded context into the LLM — delivering accurate, hallucination-resistant responses. Authentication and complete data isolation are enforced through Supabase with Row Level Security, ensuring enterprise-grade security at every layer.

<br/>
</details>

<br/>

> A real-time geopolitical intelligence platform with a live global threat dashboard, an operational command console, and an LLM-powered incident analysis co-pilot — engineered in 24 hours at a national hackathon.

<div align="center">

| Attribute | Details |
|:---|:---|
| **Stack** | React.js · FastAPI · Anthropic Claude API · Real-time data feeds |
| **Scale** | 45+ toggleable data layers across live global intelligence streams |
| **AI Core** | LLM reasoning pipeline · Threat assessment generation · Ranked response options |
| **Interface** | Interactive 3D globe · 2D/3D tactical map · Live scorecards |
| **Security** | Incident-scoped data collection · Structured LLM output validation |
| **Repository** | [![GitHub](https://img.shields.io/badge/View%20Repo-171515?style=flat-square&logo=github&logoColor=white)](https://github.com/parthgenx/w-beautiful-dark-circles) |

</div>

SENTINEL is an AI-native command intelligence platform engineered for analysts operating in high-stakes environments. The World Monitor dashboard renders a live interactive globe with 45+ toggleable data layers spanning conflict zones, military installations, cyber threats, financial signals, and commodity markets. The Strategic Console aggregates live data into a unified command picture — with Threat Level, Readiness, and Escalation Risk scorecards alongside a switchable 2D/3D tactical map. The AI Copilot powered by Anthropic Claude collects structured incident details, runs a multi-step LLM reasoning pipeline to generate threat assessments and ranked response options, and syncs results back to the console in real time. Selected for the final round of *404 Limits Not Found* hackathon.

<br/>
</details>

<details>
<summary><b>&nbsp;⬡&nbsp; JeevBandhu — Livestock Health and Safety Platform</b></summary>
<br/>

> A verified livestock marketplace with an AI-assisted disease diagnosis engine and an automated biosafety compliance system — designed for rural agricultural ecosystems.

<div align="center">

| Attribute | Details |
|:---|:---|
| **Stack** | Full-stack · Mistral 7B · Hugging Face Inference API · Digital Health Passports |
| **AI Core** | Mistral 7B symptom-to-diagnosis pipeline · Structured treatment recommendations |
| **Compliance** | Automated drug withdrawal enforcement · Medication record tracking |
| **Trust Layer** | Digital Health Passports · Verified marketplace listings · Certified health status |
| **Impact** | Empowers rural farmers with AI-guided veterinary intelligence and safe trade |
| **Repository** | [![GitHub](https://img.shields.io/badge/View%20Repo-171515?style=flat-square&logo=github&logoColor=white)](https://github.com/parthgenx/JEEV-BANDHU) |

</div>

JeevBandhu is a mission-driven platform addressing livestock health, safety, and trade integrity for Indian farmers. The Bio-Safety Engine automates compliance by tracking each animal's complete medication history and enforcing legally mandated drug withdrawal periods before any listing is approved. The AI diagnosis engine — powered by Mistral 7B via the Hugging Face Inference API — allows farmers to describe symptoms in natural language and receive structured, actionable treatment recommendations. A verified marketplace built around Digital Health Passports ensures only medically certified animals with complete health records can be listed for sale. Shipped end-to-end within 24 hours at *RepoGenesis*, a national-level IEEE hackathon.
